# SQL Data Analysis Project

## Objective
Analyze a relational database using SQL to extract insights related to
product popularity, customer behavior, and sales performance.

## Tools Used
- SQL (SQLite)
- Python
- Pandas
- Jupyter / Google Colab

## Dataset
Relational database simulating an online bookstore, including tables for
books, authors, customers, orders, and reviews.

## Key Skills Demonstrated
- Writing complex SQL queries
- JOINs across multiple tables
- Aggregations and grouping
- Translating business questions into queries
- Interpreting query results

## Project Context
This project was completed during a formal data analysis certificate
program and is presented here as a demonstration of SQL and analytical
skills.
